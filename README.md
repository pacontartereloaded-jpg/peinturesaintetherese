# Peinture Sainte-Thérèse

Site vitrine d'une équipe de peintres artisans à Sainte-Thérèse, Québec — spécialistes en restauration de boiseries, plâtre ancien et couleurs historiques.

## Stack
HTML5 + CSS3 + JavaScript vanilla. Aucune dépendance.

## Sections signature
- **Histoire / cachet** — l'approche philosophique pour les maisons d'époque
- **Spécialités** — 4 savoir-faire distincts (boiseries · plâtre ancien · plafonds & médaillons · couleurs historiques)
- **Process artisanal** — 6 étapes lentes de la visite à la dernière retouche
- **Galerie** — cartes encadrées en bois (paper-double-frame)

## Stack visuel
- Palette : beige #ede4d3 / bordeaux #7a3a3a / bois #8c6f4e
- Typo : Playfair Display (display) + Lora (body)
- Effets : paper grain SVG fixed overlay, slow reveals 1.1s, ornement ❦ comme accent
